<h1 align="center">CoRefactor</h1>
<h4 align="center">Real-Time Collaborative Python Code Editor with AI Assistance</h4>

---

<h2 align="center">Project Overview</h2>

**CoRefactor** is a cutting-edge, web-based platform designed for real-time collaborative coding. It enables multiple developers to work together seamlessly on Python code, supported by integrated AI code assistance, live chatting, and activity tracking.

This project was originally developed during a hackathon to tackle issues faced by developers working together. Its user interface is intuitive, productivity-focused, and visually appealing, making coding with teammates or learning together fast and enjoyable.

---

<h2 align="center">üí° Key Features</h2>

-   **User Authentication** ‚Äì Secure registration, login, and profile management with password hashing.
-   **Real-Time Collaboration** ‚Äì Multiple users can edit code together using Flask-SocketIO.
-   **Live Chat** ‚Äì General chat room for communication in each coding session.
-   **Activity Tracking** ‚Äì Automatic logging for joins, edits, and AI interactions.
-   **Avatars** ‚Äì Automatic avatar generation for user profiles.
-   **Code Execution** ‚Äì Run Python code with basic safety checks, share output live.
-   **AI Code Assistant** ‚Äì Ask coding questions in the editor, powered by Ollama (`llama3` model).
-   **Chat History** ‚Äì Stores both AI and user discussions per room.
-   **Room Management** ‚Äì Create, join, and leave unique code collaboration spaces.
